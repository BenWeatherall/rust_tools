# Plan Phase Prompt

Use after research is complete.

## Task

Create an implementation plan for:

> [DESCRIBE FEATURE HERE]

## Instructions

1. Read `docs/planning_docs/implementation_research.md`
2. Create `docs/planning_docs/implementation_plan.md` with:
   - Product requirements (goal, non-goals)
   - Architecture decisions with SOLID rationale
   - Atomic step checklist (each step independently verifiable)
   - Testing strategy
   - Approval gate checkbox (unchecked)

## Constraints

- Steps must map to single workspace members where possible
- Each step must include validation criteria
- Do NOT begin implementation until approval gate is checked

## Output

A completed plan awaiting human or supervisory agent approval.
