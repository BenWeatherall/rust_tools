# SR5: RPI Workflow

## Scope

Research → Plan → Implement workflow artifacts and reusable prompts.

## Delivered

- `docs/planning_docs/README.md` — developer guide
- Templates: `implementation_research.md`, `implementation_plan.md`, `adversarial_review.md`
- Prompts: `.cursor/prompts/01-research.md` through `04-adversarial-review.md`

## Gates

1. Research: no `.rs`/`.slint` edits
2. Plan: approval checkbox required
3. Implement: one atomic step at a time

## Validation

Prompts reference correct skill names and `docs/planning_docs/` paths.
